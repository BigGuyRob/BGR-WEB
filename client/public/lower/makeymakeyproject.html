Makey Makey Easy Access Application

Music Making Board Extension

Makey Makey

Junior Inventor Kit

This kit is designed to connect everyday objects to computer keys. Using a circuit board, alligator clips, and a USB cable, it uses closed loop electrical pulses to send signals the computer.

Context

Makey Makey Junior Inventor Kit was introduced to me while taking Guitar and Piano in my senior year at PTHS.
My teacher ask me to make an app that allowed her to set the sounds for each input.
She also wanted a small strap made in order to fit the alligator clips securely on her finger.

Easy Access

Standalone Executable

The application is designed to allow the selection of audio files to play when a key is registered from board.
Later to include the creation of a profile so users can save the sounds.

Programming Language

Visual Basic

Creating the application using Visual Basic allowed simplicity and efficiency in providing the application to the user.
